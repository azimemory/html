Form관련 태그

	 	Form 태그는 HTML 내에서 사용자가 입력할 수 있는 양식을 제공한다.
	 	* action 속성 : form에 입력된 값들을 전송할 url을 작성
	 	* method 속성  : form에 입력된 값들을 전송할 방식(HTTP Method)을 지정
	 	* enctype 속성 : form에 입력되어 전송 될 값들의 Mime Type
	 					 데이터에 대한 정보, 어떤 형식으로 만들어졌는지, 어떤 기능을 하는 지
	 					 ex) text/css : 해당 데이터가 문자들로 이루어져 있고 CSS기능을 하는 데이터
